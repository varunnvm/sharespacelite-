ShareSpaceLite

ShareSpaceLite is a modern platform designed to simplify the process of sharing and managing spaces. The application provides an intuitive interface for users to discover, list, and manage shared spaces efficiently.

Features

* User-friendly and responsive interface
* Secure user authentication and authorization
* Space listing and management
* Search and filtering functionality
* Real-time data updates
* Mobile-friendly design
* Fast and lightweight performance

Tech Stack

Frontend

* HTML5
* CSS3
* JavaScript

Backend

* Python
* Flask

Database

* SQLite / MySQL

Installation

1. Clone the repository

git clone https://github.com/varunnvm/sharespacelite-.git

2. Navigate to the project directory

cd sharespacelite-

3. Create a virtual environment

python -m venv .venv

4. Activate the virtual environment

source .venv/bin/activate

5. Install dependencies

pip install -r requirements.txt

6. Run the application

python app.py
